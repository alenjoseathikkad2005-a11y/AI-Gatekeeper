# Face Recognition Attendance

Simple steps:

## 1) Install Python packages
Open terminal in project folder and run:

```bash
pip install -r requirements.txt
```

## 2) Start project

```bash
cd backend
python app.py
```

## 3) Open in browser

`http://127.0.0.1:5000`

## Login
- Username: `admin`
- Password: `admin123`

## Important
- Keep model file here: `backend/models/best.pt`
- If face matches but no uniform/ID card: fine `Rs 500`
- Admin clicks `Tick Paid` in Attendance page, then attendance is marked.
